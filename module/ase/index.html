ASE Module

The ase.nf module supplies two boilerplate processes aseSP and aseMD that be used to construct processes based on ASE. A typical usage of the module is to supply a python script that defines the an ASE calculator object as the input, and make the SP and MD computation with ASE. Those usage can be easily achieved by the provided sp and md workflow. The DFTB+ module is an example of such usage.
